## 🚀 Getting started

### 🐳 Avec docker 

Pour démarrer le projet avec Docker, exécutez la commande suivante :

```sh
$ docker compose up --build
```

### 💻 En locale

Pour démarrer le projet en local, exécutez la commande suivante :

```sh
$ npm run dev
```
